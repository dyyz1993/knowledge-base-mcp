Quick Start

Up and running in 60 seconds

三步启动你的 AI 知识库服务器。

1

Install

使用 Bun 全局安装 KB-MCP 服务器。

bun add -g @dyyz1993/kb-mcp
2

Start Server

启动 HTTP + Web UI 服务器，设置你的 API Key。

KB_API_KEY=your-secret kb-mcp --http --web --port 19877
3

Connect AI Agent

在你的 AI Agent (Claude / Opencode / Cursor 等) 中配置 MCP Server。

{ "mcpServers": { "kb": { "command": "kb-mcp", "env": { "KB_API_KEY": "your-secret" } } } }
